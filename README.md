### Calendar Service
A service for handling calendar's events for a user. It's written in Kotlin. 


### How to run
- Download the jar file
- Extract into a folder
- Open IDEA IntelliJ
- jdk 17 is required
- Let the dependencies download for a few secs
- Go to the main class and run it

### Functionalities
-[x] Create a user
  - add user
  - delete user
  - get users
- [x] Create an event
  - add event
  - delete event
  - get events
